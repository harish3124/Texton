* Texton
** A Simple Text editor made with Python and PyQt6
+ Made for educational purposes

[[./screenshot.jpg]]

** Installation
*** Install Poetry
#+begin_src shell
pip install poetry
#+end_src

*** Clone this repo

*** Install Dependencies
#+begin_src shell
poetry install
#+end_src

*** Run
#+begin_src shell
poetry run python src/main.py
#+end_src
